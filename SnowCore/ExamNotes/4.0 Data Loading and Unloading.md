named file format objects : when unloading similarly formatted data on a regular schedule.

LOAD_UNCERTAIN_FILES = TRUE parameter specifically addresses the scenario where file metadata has expired beyond Snowflake's 64-day retention period    
compressed file size for loading data into Snowflake using Snowpipe - 100-250MB  

Stream :   
-  'Insert-only' streams are specifically supported for tracking changes in external tables.    


Unload : 
- Formats  are Delimited files (CSV, TSV, etc.), JSON, and Parquet.   

PUT : Uploads one or more data files from a local file system onto an internal stage. then use COPY INTO to Load     
      PUT does not support uploading files onto an external stage.    
GET : Downloads data files from internal stage to a local directory or folder on a client machine   
REMOVE : Removes files from either an external (external cloud storage) or internal (i.e. Snowflake) stage.
