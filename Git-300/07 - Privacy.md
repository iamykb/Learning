Content exclusion lets repository, organization, or enterprise administrators explicitly exclude sensitive files or directories from Copilot’s influence. This prevents Copilot from accessing these files—so it won’t suggest them, nor learn from them when generating code completions or chat responses. This is the recommended mechanism to safeguard proprietary content.
Exclusion to be set in organization settings and / or Enterprise settings.

GitHub Advanced Security (GHAS) secret scanning helps prevent the accidental inclusion of sensitive credentials
